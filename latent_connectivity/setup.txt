Requirements:
-------------
python (version 3.x)
numpy
pandas
rpy2
R (version 3.2+)
psych


Installation recommendation:
----------------------------
# Use anaconda to set up an environment with the dependencies
# Note: Make sure anaconda packages are up-to-date
#conda update conda
#conda update anaconda
conda update --all
# create anaconda environment 'r-env'
conda create --name r-env -c r python=3.4 anaconda rpy2 r-psych
# activate anaconda environment
source activate r-env
